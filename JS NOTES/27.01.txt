LOCAL STORAGE

Obs: armazena os dados salvos do navegador, em um diretorio local da maquina para quando a pagina for recarregada, os dados não serem perdidos, o localStorage apenas armazena valores em String.

LocalStorage: variavel global fornecida pelo browser para acessar os métodos de armazenamento local.

localStorage.setItem('<Key>', '<value>'): metodo que armazena um valor String no localStorage com um identificador definido pelo parâmetro key.

localStorage.getItem('<Key>'): retorna o valor da localStorage de acordo com a key passada por parâmetro

localStorage.removeItem('<Key>'): remove um elemento do localStorage

localStorage.clear(): remove todos os elementos do localStorage


JSON: (javascript Object Notation), variavel global fornecida pelo browser que converte elementos em strings e converte de volta.

JSON.stringify(): converte Objetos em Strings

JSON.parse ('<Objeto>'): converte Strings em Objetos
